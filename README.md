# WLED Wood Clock
## Introduction
Heyya, thanks for actioning the clock. Jarm built the clock, i only modified the firmware of the controller and set up the controller. So if you need help with the clock, its better to first ask him.
## Description
The Clock is built with two LED-Strips inside, one creating the clock, the other one creating the outside. The first Clock segement consists out of the leds: `0-65`, the second outer segment is: `65-end`.
The clock is a usermod, which has been compiled into the default [WLED Firmware](https://github.com/wled/WLED), its called [Seven Segment Display Reloaded](https://github.com/wled/WLED/tree/main/usermods/seven_segment_display_reloaded).

I modified the original source code, to allow having two strips and the clock on the same controller, my new source code can be found here [https://github.com/forest-cat/WLEDUsermods](https://github.com/forest-cat/WLEDUsermods).
